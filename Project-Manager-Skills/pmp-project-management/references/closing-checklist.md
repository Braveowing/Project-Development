# 项目收尾检查清单

## 交付物验收

- [ ] 所有交付物已按范围基准和验收标准完成
- [ ] 每个交付物已获得客户/发起人正式验收签字
- [ ] 遗留缺陷/问题已记录并约定处理方案

## 移交与过渡

- [ ] 运营团队已接收并确认具备承接条件
- [ ] 必要的培训已完成
- [ ] 操作文档、维护手册已交付
- [ ] 知识转移会议已完成

## 财务收尾

- [ ] 所有合同/采购已完结并结算
- [ ] 项目实际成本已汇总并与预算对比
- [ ] 剩余预算/储备已归还

## 文档归档

- [ ] 项目管理计划（最终版）
- [ ] 范围、进度、成本基准（最终版）
- [ ] 风险登记册（关闭状态）
- [ ] 变更日志（全部已关闭）
- [ ] 问题日志（全部已关闭）
- [ ] 状态报告（全部）
- [ ] 合同与采购文件

## 经验教训

- [ ] 召开经验教训总结会议
- [ ] 记录有效的做法（值得复用）
- [ ] 记录问题和改进建议（值得避免）
- [ ] 将经验教训更新到组织知识库

## 团队

- [ ] 团队成员绩效评估已完成
- [ ] 团队成员已正式释放并可分配到新项目
- [ ] 团队表彰/感谢已完成

## 组织过程资产更新

- [ ] 模板更新（如发现改进点）
- [ ] 流程改进建议已提交
