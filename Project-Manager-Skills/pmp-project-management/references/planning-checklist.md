# 规划检查清单

项目规划阶段需完成以下产出物。根据项目规模和复杂度，可适当裁剪。

## 必需（所有项目）

- [ ] **范围基准**
  - 项目范围说明书（做什么、不做什么）
  - WBS（工作分解结构）
  - WBS 词典（各工作包的定义和验收标准）
- [ ] **进度基准**
  - 活动清单与依赖关系
  - 里程碑计划
  - 甘特图/进度表
- [ ] **风险登记册**
  - 已识别风险及评估
  - 风险应对策略
- [ ] **相关方登记册**
  - 相关方信息、期望、参与策略

## 推荐（中型及以上项目）

- [ ] **成本基准** — 预算估算与储备
- [ ] **质量管理计划** — 质量标准与验收标准
- [ ] **资源管理计划** — 团队结构与职责分配（RACI）
- [ ] **沟通管理计划** — 沟通频率、渠道、受众
- [ ] **变更管理计划** — 变更提交流程与审批权限

## 可选（大型/复杂项目）

- [ ] **采购管理计划** — 外部采购策略
- [ ] **相关方参与计划** — 详细参与策略
- [ ] **项目管理计划汇总文档**

## 方法论适配

### 预测型项目补充
- 详细的阶段门（Phase Gate）定义
- 完整的变更控制委员会（CCB）机制

### 适应型项目补充
- 产品待办列表（Product Backlog）
- 迭代/冲刺计划模板
- 回顾会议议程

### 混合型项目补充
- 明确哪些模块用预测型、哪些用适应型
- 两种方法的集成点定义
